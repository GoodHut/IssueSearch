 URL : "https://github.com/ChiChou/koa-sqlite3-session/issues/1" TITLE : ttl should be in ms BODY : the documentation for koa-generic-session shows milliseconds ms are supposed to be used, instead of seconds. for example, this should have a ttl of 1 day, but sqlite3store gives it a ttl of 1,000 days: app.use session { store: new sqlite3store 'filename.db' , ttl: 86400000 // ms } ;
