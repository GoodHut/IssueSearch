 URL : "https://github.com/wpninjas/ninja-forms/issues/2988" TITLE : database charset collation BODY : wordpress tables, ie wp_options , are setup with the utf8mb4 character set collation as of v4.2 . ninja forms is currently installing tables based on the character set collation defined by $wpdb . this creates a situation where the character set collations are different between ninja forms and wordpress .
