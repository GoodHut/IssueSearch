 URL : "https://github.com/ESMCI/cime/issues/1584" TITLE : add a case.qstatus command BODY : case.submit provides a flexible wrapper around the various qsub, brun, llsubmit, etc. commands that various schedulers use. to unify the cime user experience, we should add a case.qstatus command that will execute the appropriate qstat, llq, whatever-slurm-uses command to show the status of the submitted jobs in the queue.
