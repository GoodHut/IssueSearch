 URL : "https://github.com/EdgarSeca/EdgarMendes_pari/issues/1" TITLE : lista de ficheiros BODY : main.c - só tem o programa principal callbacks.c - contem as callbacks dos widgets, e handlers de sinais, timers e timeouts. myf.c - contém todas as restantes funções auxiliares. myf.h - header file do projeto variáveis globais, definições de tipos, inclusão de outras headers, etc. . prototypes.h - header file com os protótipos das funções gerado com genproto.sh ou similar . mydr2.glade - ficheiro glade com a interface gráfica a usar.
