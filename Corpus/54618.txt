 URL : "https://github.com/nightcoder21/art/issues/3" TITLE : group selectors in css BODY : simplify! consider using group selectors for any repeated styles. for example you could use: h1, brush, span, frame, paint { font-family: helvetica; font-weight: bold; } brush, frame, paint { font-size: 32px; color: white; } then delete the styles from the individual selectors. this practice will help your css to be more concise.
