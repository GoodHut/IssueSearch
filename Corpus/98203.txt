 URL : "https://github.com/luciopaiva/witchcraft/issues/5" TITLE : in-memory cache of scripts BODY : cache to avoid going to the disk all the time, which can significantly slow down page loading. this is a more involved enhancement, since the extension would have to know when to invalidate cached scripts. the server could make use of node's file watch api https://nodejs.org/docs/latest/api/fs.html fs_fs_watch_filename_options_listener to know when to refresh them.
