 URL : "https://github.com/mercyblitz/dubbo/issues/16" TITLE : dubbo rest 支持调研 BODY : dubbo rest 支持主要有两部分的考虑： 1. http web 服务 servlet：绝对大多数 web 应用基于 java servlet 规范开发，因此 可以考虑 利用servlet 3.0 +的自动装配 netty ：目前 dubbo 直接依赖了 netty 框架，也可以考虑支持嵌入式 netty web 容器 2. rest 处理 jax-rs： jax-rs 是 jsr rest 标准，目前参考实现 jersey 提供了完整的支持，比如：netflix 处理 rest 也基于此，支持标准是必然的趋势 spring web mvc ：spring web mvc 基本上属于事实的标准，经过了大量的线上应用测试，各方面表现良好，没有不支持的理由
