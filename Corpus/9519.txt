 URL : "https://github.com/levelp/JavaPatterns/issues/11" TITLE : мост bridge BODY : структура, позволяющая изменять интерфейс обращения и интерфейс реализации класса независимо. пример: архитектура java awt полностью основана на этом шаблоне — иерархия java.awt.xxx для хэндлов и sun.awt.xxx для реализаций.
