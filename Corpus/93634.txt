 URL : "https://github.com/ifrost/afterwriting-labs/issues/67" TITLE : allow multiple empty lines BODY : by default additional empty lines between text are ignored. fountain syntax, however, allows to add extra separators for vertical alignment https://fountain.io/syntax section-br . to keep the current behaviour an option should be added to settings ignore extra empty lines that would be set to false by default. when the option is disabled, double space between scenes should still play nicely, adding additional separator only if needed. note: at the moment, adding extra empty lines can be achieved by using non-breaking space double space
