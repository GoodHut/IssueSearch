 URL : "https://github.com/algolia/react-instantsearch/issues/593" TITLE : migration to rollup BODY : today we use webpack for bundle the umd build & a custom uglifyjs script to minify and generate the sourcemap. we could probably use rollup instead for three main reasons: - remove the hard dependency on webpack & uglifyjs script for the build - probably save a bit of kb on the payload of the library - simplify the build process
