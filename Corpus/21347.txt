 URL : "https://github.com/jaoyina/kms/issues/23" TITLE : f450a reset telnet BODY : 先捅复位孔释放telnet功能。 运行中输入：telnet 192.168.1.1 用户名：root 密码：zte521 输入 sendcmd 1 db p cltlmt 查看一下当前的用户数量 输入 sendcmd 1 db set cltlmt 8 max 99 修改为99 输入 sendcmd 1 db p pdtctuserinfo 查看授权状态 输入 sendcmd 1 db set pdtctuserinfo 0 status 0 修改为已注册 输入 sendcmd 1 db set pdtctuserinfo 0 result 1 修改为已授权 输入 sendcmd 1 db save 保存 reboot 重启 原理： 中兴从很早就开始不支持捅复位孔复位了，捅了复位孔只能释放telnet功能，相关配置不会掉，但是会掉授权状态。所以先修改限制台数之后再修复授权状态，重启之后就可正常了。
