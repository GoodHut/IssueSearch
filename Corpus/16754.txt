 URL : "https://github.com/meircif/mr-dev/issues/60" TITLE : loop type: repeat BODY : in the fibonacci example, we can see a common programming language pattern: func fibonacci copy int n : copy int res var int prev 1 res := 0 for _ in n var int sum res + prev prev := res res := sum notice that the for loop is not really used for its variable. i suggest a repeat loop type that just accepts a number to repeat to loop: func fibonacci copy int n : copy int res var int prev 1 res := 0 repeat n: var int sum res + prev prev := res res := sum
