BODY : ref: struct https://ruby-doc.org/core-2.4.1/struct.html struct is a handy and simple way to dynamically create new class. it can be served as temporary data structure, or an internal class data. for example: ruby foo = struct.new :bar => <class:0x007f8ecf09a0f0> a = foo.new 1 => <struct bar=1> a.bar => 1 or even simpler, open struct : ruby a = openstruct.new { foo: 1, bar: 2 } or openstruct.new foo: 1, bar: 2 a.foo => 1
