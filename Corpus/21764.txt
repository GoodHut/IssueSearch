 URL : "https://github.com/DICKLEE/python/issues/1" TITLE : gmail using python BODY : 使用 python 發送 gmail 信件 參考連結一 ： http://stackabuse.com/how-to-send-emails-with-gmail-using-python/ 參考連結二 ： https://support.google.com/accounts/answer/185833 到參考連結二產生應用程式密碼，共16個字元 python import smtplib user = 'me@gmail.com' pwd = '16個字元密碼' to = 'weihung0714@gmail.com' msg = '''//from: me@gmail.com subject: this is a test mail hello the test is working! ''' server = smtplib.smtp 'smtp.gmail.com',587 server.ehlo server.starttls server.login user, pwd server.sendmail user, to, msg server.quit
