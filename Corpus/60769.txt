 URL : "https://github.com/wking/fsf-api/issues/1" TITLE : add fsffree designation BODY : please add a field on the json representing if an fsf license is considered free / libre. the licenses listed on the free software foundation license list https://www.gnu.org/licenses/license-list.en.html have the following designations: 1. free licenses compatible with the gnu gpl 2. free licenses compatible with the fdl 3. free licenses incompatible with the gnu gpl and fdl 4. nonfree licenses 5. licenses for work stating a viewpoint the free / libre designation would be a license identified as 1, 2, or 3 in the above list.
