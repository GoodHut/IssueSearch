 URL : "https://github.com/PySlurm/pyslurm/issues/93" TITLE : compiling on ubuntu BODY : after a couple of installing errors, here's some suggestion to complete compilation on ubuntu 14.04 build slurm 17.02 first git clone git@github.com:schedmd/slurm.git cd slurm git checkout slurm-17.02 mkdir build cmake -dcmake_install_prefix=~/.local .. make install git clone git@github.com:pyslurm/pyslurm.git cd pyslurm python setup.py build --slurm=~/.local/ python setup.py install it finally works for me
