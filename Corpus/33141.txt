 URL : "https://github.com/music-encoding/music-encoding/issues/420" TITLE : add @class attribute BODY : @class provides a mechanism for classifying an element according to an established, formal taxonomy. it has a different function from @type , which is for ad hoc extension and restriction of the schema. @class may contain one or more uris which denote classification terms that apply to the entity bearing this attribute. a fragment identifier e.g., ' term01 points to a classification term recorded in the mei header. a schematron rule will ensure that the value in @class corresponds to the @xml:id attribute of a <term> element or is an external url, i.e., one that matches '^ a-z +://|\.{1,2}/ '. @class will be available anywhere @type is.
