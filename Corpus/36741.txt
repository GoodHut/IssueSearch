 URL : "https://github.com/Valkryst/Schillsaver/issues/24" TITLE : implement job dialog BODY : acceptance criteria user requirements: user can open a menu to add files to the job. user can a file from the job. user can delete all files from the job. user can specify whether the job is for encoding or decoding. user can specify a name for the job. the name must be an auto-generated uuid if the user does not specify a name. user can specify a description for the job. user can view a time estimate for how long the encoding or decoding will take. user can select an output folder.
