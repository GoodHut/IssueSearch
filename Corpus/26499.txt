 URL : "https://github.com/theia-ide/theia/issues/165" TITLE : basic git extension BODY : a git extension shall allow the user to interact with a git repository of the file system. the user should be able to perform the following git commands through a graphical user interface: - fetch - pull - merge - commit - push a dedicated git view will list all changed files in sections depending on whether they are staged or un-staged. double-clicking a file entry will open a diff editor showing the changes. an action allows staging respectively un-staging the changes of a file. the diff editor allows to further modify the local file copy.
