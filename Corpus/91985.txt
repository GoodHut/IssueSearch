 URL : "https://github.com/dojo/shim/issues/109" TITLE : use of downleveliteration BODY : enhancement we should consider using the compiler flag downleveliteration . our symbol shim should be sufficient to allow typescript to down emit for ... of structures on any iterators. this means that we would not have to use the iterator/forof function to replicate iteration and more widely adopt iterables throughout the code base as well as allow downstream authoring with iterables .
