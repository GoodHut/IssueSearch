 URL : "https://github.com/driverdan/node-XMLHttpRequest/issues/145" TITLE : use with aws lambda BODY : if you want to use this package with aws lambda, you need to modify the two lines: var contentfile = .node-xmlhttprequest-content- + process.pid; var syncfile = .node-xmlhttprequest-sync- + process.pid; with var contentfile = /tmp/node-xmlhttprequest-content- + process.pid; var syncfile = /tmp/node-xmlhttprequest-sync- + process.pid; because you can write only in the /tmp/ directory
