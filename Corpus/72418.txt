 URL : "https://github.com/jehrhardt/lambdaapp/issues/3" TITLE : build app for production BODY : requirements when a new deployment is created, a build will also be created. the build will be executed before the deployment. the build has the following properties: app stage app_image finish_time finish_state the deployment gets an additional reference to the build. the build is done by running build commands on the container scheduler. the app_image should also contain the version based on the git commit id. contraints the container scheduler can be mocked so far. the build commands are read from a configuration in the git repository.
