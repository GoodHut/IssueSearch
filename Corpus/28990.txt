 URL : "https://github.com/openhardwaremonitor/openhardwaremonitor/issues/1041" TITLE : incorrect bus speed BODY : i have run ohm on a couple different dell systems. the bus speed value/max/min all report 100 mhz, and never change. after digging around with powershell, i think you are using the cpu external clock value, for finding the cpu clock bus speed. i think what you are really looking for is the physical memory configured clock speed, 2133 mhz.
