 URL : "https://github.com/ressec/athena/issues/36" TITLE : create a command terminal BODY : purpose the command terminal embeds a command interpreter for text commands entered on the console. it provides some services to register some command definitions. its constructor is taking as parameter an xml command file containing command definitions that are registered just after creation. when a command is interpreted identified and validated then the terminal underlying thread is paused to avoid entering new commands until the response has been received and processed. only then, the underlying terminal thread is resumed. see: 11
