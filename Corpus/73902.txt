 URL : "https://github.com/afourmy/eNMS/issues/17" TITLE : different type of script BODY : let the user choose the type of script: - normal: no yaml file - j2 template: a yaml file used for all target devices - per-node template: the yaml file should have, as first-level keys, the name of the target devices r1: values r1... r2: values r2... ... rn: values rn... for each hostname, the jinja2 template uses the specific values stored in the yaml file.
