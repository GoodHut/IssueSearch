 URL : "https://github.com/mutability/dump1090/issues/203" TITLE : config for apache 2.4 BODY : if you have an existing apache 2.4 install and want to see dump1090 at http://localhost/dump1090, save the following lines to /etc/apache2/sites-available/dump1090.conf and run sudo a2ensite dump1090 afterwards: alias /dump1090/data /run/dump1090-mutability <directory /run/dump1090-mutability > require all granted </directory> alias /dump1090 /usr/share/dump1090-mutability/html <directory /usr/share/dump1090-mutability/html > options indexes followsymlinks multiviews allowoverride none require all granted directoryindex gmap.html </directory>
