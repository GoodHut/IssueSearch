 URL : "https://github.com/ZerionSoftware/website/issues/22" TITLE : server signature test BODY : by default, the apache webserver sends http headers with some information about your server version, operating system, modules installed, etc. these informations can be used by hackers in order to exploit vulnerabilities specially if you are running an older version . these information can be hidden or changed with very basic configurations. open apache?s configuration file httpd.conf or apache.conf and search for serversignature. if you find it, edit it to: serversignature off servertokens prod if you don't find it, just add these two lines at the end of the file. note that, after you modify the configuration file, you must restart the apache server.
