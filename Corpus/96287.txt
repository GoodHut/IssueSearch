 URL : "https://github.com/DataDog/dogstatsd-ruby/issues/57" TITLE : error for frozen tag array BODY : when calling any datadog::statsd metric interface, if a frozen array is passed to the tags: parameter, an error is raised. this is the code responsible in datadog::statsd : def send_stats stat, delta, type, opts={} ... tag_arr = opts :tags .to_a tag_arr.map! { |tag| t = tag.dup; escape_tag_content! t ; t } ... end if opts :tags is a frozen array, then tag_arr is also a frozen array, causing map! to error.
