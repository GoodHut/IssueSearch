 URL : "https://github.com/unispeech/asterisk-unimrcp/issues/11" TITLE : --libdir is ignored BODY : for centos 7/x86_64, the correct path for 64-bit libraries is /usr/lib64, not /usr/lib: ./configure --libdir=/usr/lib64 ... report modules install path...........: /usr/lib/asterisk/modules ... make && make install ... libraries have been installed in: /usr/lib/asterisk/modules thus, after installation, one has to manually move the libs out of /usr/lib/asterisk/modules/ into /usr/lib64/asterisk/modules/ in order for asterisk to load unimrcp.
