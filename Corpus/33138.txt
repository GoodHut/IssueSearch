 URL : "https://github.com/Backblaze/B2_Command_Line_Tool/issues/327" TITLE : add --nosleep option BODY : currently b2 process doesn't prevent computer from getting in sleep also called stand by mode. when using sync command for example which can last for hours it can be useful to disable sleep, and enabling it again when the command is done. this could be activated on a set of b2 commands using a flag like --nostandby or --nosleep .
