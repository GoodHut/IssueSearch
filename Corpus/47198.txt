 URL : "https://github.com/vdaviot/RTv1/issues/1" TITLE : obj files in .rt file BODY : hello, when you are loading a scene form a .rt file, obj and mtl files need to be loaded and stored in the final scene's linked list. for example, this scene will load the entire falcon falcon milenium with it's mtl file referenced from in the obj file : scene2: mehsobject: obj: falcon_milenium.obj pos: 0 10 0 rot: 0 150 0 light1: type: light pos: 0 0 0 intensity: 10
