 URL : "https://github.com/beccap7/prj-rev-bwfs-dasmoto/issues/1" TITLE : id's vs classes BODY : https://github.com/beccap7/prj-rev-bwfs-dasmoto/blob/master/dasmotoart/index.html l8-l29 it seems you assigned your price elements id's, however they should be classes since you are using the same price class for multiple elements to assign them the same stylings. id's are better suited for unique, individual elements while classes are better for when you want to give a group of elements the same styling. here is a great resource on the differences between classes and id's: https://css-tricks.com/the-difference-between-id-and-class/
