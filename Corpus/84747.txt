 URL : "https://github.com/red5pro/streaming-html5/issues/108" TITLE : do not use babel-polyfill BODY : when including your sdk in an app that already uses babel-polyfill the loading of sdk is not successful. based on https://github.com/babel/babel/issues/4019 and https://babeljs.io/docs/usage/polyfill/ details libraries should not use babel-polyfill. repo steps: - load babel-polyfill - load red5pro-sdk expected result: - page and sdk are working actual result: - uncaught error: only one instance of babel-polyfill is allowed
