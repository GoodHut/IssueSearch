 URL : "https://github.com/robolectric/robolectric/issues/3464" TITLE : cookies are duplicated BODY : description use the cookie manager - add a cookie on the same domain with a different value and it will appear twice. android doc states this should not be the case - already tested manually steps to reproduce 1. take cookiemanager 2. setcookie domain.com , cookie=1 ; 3. setcookie domain.com , cookie=2 ; robolectric & android version in android you would then see a cookie with the value cookie=2. in robolectric you see cookie=1; cookie=2 - which is wrong !
