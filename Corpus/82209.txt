 URL : "https://github.com/awslabs/s2n/issues/498" TITLE : add more static analyzers BODY : this issue should track our work on adding more static code analyzers. there are lots of static code analyzers for c code, a good starting point to compare them is the list provided by sei cert c coding standard here: https://www.securecoding.cert.org/confluence/display/c/ee.+analyzers static code analyzers added to s2n: - cbmc - x clang - codesonar - x cppcheck - x ctverif for constant time functions - eclair - edg - framac - x gcc - klocwork - x kwstyle - ldra - parasoft - polyspace - prqa - rose - x saw for hmac code - sonarqube - splint
