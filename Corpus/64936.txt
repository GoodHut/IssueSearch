 URL : "https://github.com/hexojs/hexo/issues/2467" TITLE : http request via helper BODY : wonder if it's possible to send http requests inside hexo helpers. the following is a simple example using the given http request https://github.com/request/request package i believe it's hexo's default dependency to send get request to the github api. any thoughts on why this is not working or how to unit test it inside one of those template files? javascript / getrepos @description get github public repos of the given author. @example getrepos author ; / hexo.extend.helper.register 'getrepos', function author { const root_url = 'https://api.github.com'; let request = require 'request' ; let url = root_url + '/users/' + author + '/repos'; request url, function error, response, body { return response; } } ;
