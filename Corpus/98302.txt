 URL : "https://github.com/slamdata/slamdata/issues/1960" TITLE : inline chart type switcher BODY : the inline chart type switcher is an icon in a chart view mode that, when pressed, shows all the options that have been setup by the author. the consumer can choose any of the chart options, which causes the chart type to switch to the specified chart type, as configured by the author of the deck. if there is only one chart type that has been setup by the author, then the icon does not appear. see here for mocks. https://docs.google.com/a/slamdata.com/drawings/d/18qy-nwkn3gvrnjmigsajnqpboqzq3yieyyu8vw3zg2y/edit?usp=sharing
