 URL : "https://github.com/riderkick/FMD/issues/802" TITLE : issue with old favorites BODY : old favorites when moved to database. have problem with: - websitelink - link this causes problem that when we click on favorites and view manga info we see manga as if is in favorites. but if we search this manga and then click. then it isn't in our favorites. example error in database : mangafoxhttp://mangafox.me/manga/... insted of mangafox/manga/... http://mangafox.me/manga/... insted of /manga/... the same problem exist in downloadedchapters.db file to. solution: update downloadedchapters set websitelink = replace websitelink, 'http://mangafox.me', '' where websitelink like '%http://mangafox.me%'; update favorites set websitelink = replace websitelink, 'http://mangafox.me', '' where websitelink like '%http://mangafox.me%'; update favorites set link = replace link, 'http://mangafox.me', '' where link like '%http://mangafox.me%'; update downloads set link = replace link, 'http://mangafox.me', '' where link like '%http://mangafox.me%';
