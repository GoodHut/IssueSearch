 URL : "https://github.com/ststeiger/DAL/issues/1" TITLE : this is a dbal, not dal BODY : dbal = database abstraction layer https://en.wikipedia.org/wiki/database_abstraction_layer > database abstraction layer is an application programming interface which unifies the communication between a computer application and databases such as sql server, db2, mysql, postgresql, oracle or sqlite. dal = data access layer https://en.wikipedia.org/wiki/data_access_layer > data access layer dal in computer software, is a layer of a computer program which provides simplified access to data stored in persistent storage of some kind, such as an entity-relational database.
