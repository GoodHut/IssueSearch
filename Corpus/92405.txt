 URL : "https://github.com/winstonjs/winston-daily-rotate-file/issues/106" TITLE : winston 3.0 compatibility BODY : winston 3.0-rc introduced some breaking changes to the transport plugin model. as a result, the current version of winston-daily-rotate-file does not work with winston >= 3. the current workaround is to avoid upgrading to winston 3.0. however, a fix is being worked on to allow for compatibility with current and previous versions of winston.
