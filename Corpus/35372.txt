 URL : "https://github.com/PovelikinRostislav/client_server_app/issues/58" TITLE : download result file BODY : скачивание результирующего файла: 1. клик по таске в очереди. 2. открывается меню manage task. 3. клик по кнопке download result. 4. открывается диалог с выбором пути для сохранения. 5. нажимаем save. 6. формируется request: type= dlresult , taskid= ... 7. на сервере по taskid ищем output file в бд. 8. кладем файл в response и отправляем на клиента. 9. на клиенте сохраняется файл в выбранное место.
