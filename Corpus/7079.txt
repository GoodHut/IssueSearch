 URL : "https://github.com/ucscXena/ucsc-xena-client/issues/153" TITLE : data sets pages BODY : currently the data sets pages tool on xena browser only report loading error when the genomics data sets itself has an error, but it does not check if the associated probemap loading also generated errors. this has caused a few problems for users that they thought their data is properly loaded to xena hub, while there is in fact a formatting problem with the probemap file, which leads to failure of visualization of the data on the xena browser. check loading error on the associated probemap file, and reflect that on data sets pages
