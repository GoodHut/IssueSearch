 URL : "https://github.com/mbrandau/simple-chat/issues/47" TITLE : create a chatcomponent BODY : use ng g c chat/chat to create the component in a chat folder. the component should contain all messages that are sent. messages by the current user should be on the right side and those from other users on the left. both of them should be in boxes. join and leave messages should be centered. create a message model with a type string and a data any value. the component should have an array containing all those messages.
