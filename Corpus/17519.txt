 URL : "https://github.com/Microsoft/vscode/issues/20718" TITLE : preserve eol type BODY : - vscode version: 1.9.1 - os version: windows 10 we already have { files.eol : \r or } to set default eol type. i suggest splitting this setting into two parts: - files.eol : no change . - files.preserveeol : if set to true , eol of old files will be preserved. if set to false , eol of old files will be set according to files.eol . new files will still follow files.eol .
