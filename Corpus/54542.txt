 URL : "https://github.com/JJoriping/KKuTu/issues/179" TITLE : gm휘장 얻는 방법 BODY : server\lib\sub\global.json 파일을 수정하고 성공적으로 gm권한을 얻었습니다. 그 다음으로 휘장을 얻으려 하는데 psql로 update users set box = '{ b1_gm :1}' where _id = '116048762675434286450'; 을 치라는 이슈를 보고 윈도우 검색창에 나오는 psql를 실행 후, update users set box = '{ b1_gm :1}' where _id = '116048762675434286450'; 를 쳤다만 다음에는 데이터베이스 postgres 라 뜨네요 예시라도 알려주시면 감사하겠습니다.
