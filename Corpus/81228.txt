 URL : "https://github.com/fpcorso/quiz_master_next/issues/516" TITLE : weighted correct answers BODY : general description a user would like some questions to be worth more than others in the final score by setting a weight to each question. feature request the user's example had 5 questions worth 5% each and then 5 questions worth 15% each. so, if a user gets one of the first 5 incorrect, he or she would receive a 95%. however, if the user gets on of the last 5 incorrect, he or she would receive an 85%.
