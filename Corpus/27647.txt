 URL : "https://github.com/alekseynemiro/nemiro.oauth.dll/issues/29" TITLE : don't force tls 1.0 BODY : hi, in executerequest nemiro.oauth is forcing the use of tls 1.0 as security protocol. this make impossible to connect to web server that only support tls 1.1/1.2 enabled failing with an existing connection was forcibly closed by the remote host . one quick fix is simply replacing that line with: servicepointmanager.securityprotocol |= securityprotocoltype.tls; so that it will allow tls 1.0 while and still keeping the other protocols enabled. but i think that don't enabling tls 1.0 at all would be the a wiser way to go.
