 URL : "https://github.com/arcticicestudio/nord-gnome-terminal/issues/8" TITLE : new color configurations BODY : gnome terminal version 3.20+ now supports more color theme options: - use-theme-transparency boolean - use-transparent-background boolean - highlight-colors-set - flag to indicate that custom highlight colors have been set: - cursor-foreground-color string - cursor-background-color string - highlight-foreground-color string - highlight-background-color string
