 URL : "https://github.com/Igalia/mesa-dockerfiles/issues/8" TITLE : build txc_dxtn conditional BODY : libtxc-dxtn uses the patented s3 texture compression algorithm. therefore, we don't want to use this library but it is still possible through setting the use_txc_dxtn variable to yes in the travis web ui. according to wikipedia, the patent expires on october 2, 2017: https://en.wikipedia.org/wiki/s3_texture_compression patent
