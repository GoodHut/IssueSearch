 URL : "https://github.com/akka/alpakka/issues/232" TITLE : contraband for data types BODY : quite a few connectors require some sort of data types for configuration. in the current situation these data types lack uniformity across connectors and in quite a few cases java api is an afterthought or being left out all together. contraband http://www.scala-sbt.org/contraband/ is a description language for data types with a generator of pseudo case classes. generated classes have java friendly api with methods and also are evolution friendly hidden copy methods and constructors .
