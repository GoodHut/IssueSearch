 URL : "https://github.com/chouzeJkl/lintcode/issues/3" TITLE : a + b 问题 BODY : 给出两个整数a和b, 求他们的和, 但不能使用 + 等数学运算符。 注意事项 你不需要从输入流读入数据，只需要根据aplusb的两个参数a和b，计算他们的和并返回就行。 说明 a和b都是 32位 整数么？ 是的 我可以使用位运算符么？ 当然可以 样例 如果 a=1 并且 b=2，返回3 思路：使用位运算 / @param a: an integer @param b: an integer @return: the sum of a and b / const aplusb = function a, b { if a==0 return b; if b==0 return a; let sum,i; i =a^b; sum= a&b <<1; return aplusb sum,i ; }
