 URL : "https://github.com/rancher/rancher/issues/7661" TITLE : agent is leaking fds BODY : agent is leaking file descriptors when a docker event is created. steps to reproduce that: 1. launch rancher server with one host. 2. ssh into the host, find the pid of agent, and do lsof -a -p $pidofagent to find the fds agent is using. 3. launch several container in rancher, compare the number of fds used by agent before and after starting containers. 4. should be a significant amount of increment. 4 or 5 per container
