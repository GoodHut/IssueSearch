 URL : "https://github.com/lc-soft/LCUI/issues/95" TITLE : css add @font-face BODY : at present, lcui need to load the font file to know the font family name, if the one-time loading too many font files will slow down the application launch speed, and manually write c or xml code to load the font file is very troublesome. therefore, we need to let the css parser support record the font name and file path, so that we can load the font file by specifying the font family name. css @font-face { font-family: 'my font'; src: '/path/to/font/file'; }
