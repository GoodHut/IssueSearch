 URL : "https://github.com/davisb10/500pxSharp/issues/5" TITLE : create gallery endpoint BODY : gallery endpoint must be able to handle the following: get - get all galleries for a given user - get a specific gallery for a user - get all photos in a given gallery put - update the metadata for a given gallery - adds, removes, or updates the order of items in the gallery. post - creates a new, empty gallery owned by the given user. delete - delete a given gallery
