 URL : "https://github.com/BaseXdb/basex/issues/1458" TITLE : restxq: status code 410 BODY : if the restxq annotation %rest:single is used, and if a request is canceled, 410 will be returned by the web server see http://docs.basex.org/wiki/restxq query_execution . the chosen code causes problems with some browsers, because 410 indicates that the “resource is no longer available” https://tools.ietf.org/html/rfc7231 section-6.5.9 , so a different code should be chosen. new proposal: 460 proprietary, as existing codes don’t really match .
