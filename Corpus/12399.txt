 URL : "https://github.com/fvsch/devtools-svg/issues/2" TITLE : panel icon style BODY : in firefox stable if you provide a white icon it gets flipped to black by the devtools css when needed light theme, inactive tab . nightly seems to have given up on this technique, so now there is no way to have an icon that is either black or white depending on the theme and status active tab or not . somewhat related: https://bugzilla.mozilla.org/show_bug.cgi?id=1391980 looks like the only solution for this and for chrome compatibility is to have an icon with a solid background e.g. like the react devtools . -__-
