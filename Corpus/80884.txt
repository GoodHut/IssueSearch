 URL : "https://github.com/pagarme/robrowser/issues/5" TITLE : use meow as a cli helper BODY : meow https://github.com/sindresorhus/meow is a cli helper, so it makes easier for user to get help and see the accepted parameters. i think most of the parameters can have default values but some of them can be override by cli parameters. for instance: --config: accepts a path to look for a config file, defaults to ./robrowser --screenshot-folder alias: -s : output folder to save screenshots, should override values in the config file -concurrency alias: -c : receive number of concurrent jobs
