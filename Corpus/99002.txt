 URL : "https://github.com/louthy/language-ext/issues/312" TITLE : lst.removeall doesn't work BODY : using the latest stable nuget package 2.1.28 . the following simple example reproduces it: csharp var test = list 1, 2, 3, 4, 5 ; console.writeline string.join , , test.removeall x => x % 2 == 0 ; // output: 1, 2, 3, 4, 5 compared to the equivalent approach using linq: csharp var test = list 1, 2, 3, 4, 5 ; console.writeline string.join , , new lst<int> test.where x => x % 2 != 0 ; // output: 1, 3, 5
