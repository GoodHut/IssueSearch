 URL : "https://github.com/dagcoin/dagcoin/issues/143" TITLE : fee solution: exchange fee BODY : a node publishes its intention to work as trader as well as the fee it will apply for the service. the fee can be set in the settings beneath the funding node switch. it should be, for the moment, a text box with an integer validation. this information including updates is shared with the discovery service should be stored in the database and updated with a message. the fee is applied on top of the conversion: let's say an unit costs 500 bytes to be generated, which translate to 0.005 dag and the fee is 0.001 dag. in this case the end-user will pay 0.006 dags for the 500 bytes.
