 URL : "https://github.com/devmynd/jsonb_accessor/issues/87" TITLE : feature enum support BODY : hi! i am interested in developing a sort of enum support for jsonb fields. the use case: a jsonb field key should be allowed to have one of a predefined set of string values. pseudocode of dsl: rb class product jsonb_accessor :data, status: :string, enum: :available, :en_route, :unavailable , default: :available end p = product.new p.status => available p.status = en_route => en_route p.status => en_route p.data => { status => en_route } p.status = discontinued => runtimeerror, status discontinued not supported, try :available, :en_route, :unavailable .
