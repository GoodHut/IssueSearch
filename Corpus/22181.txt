 URL : "https://github.com/Monstreline/prj-rev-bwfs-dasmoto/issues/4" TITLE : inheritance in css BODY : simplify! i would consider removing all instances of font-family: helvetica other than the instance on your body selector. because font-family is an inherited property there will be no need to specify the font-family property on children of the <body> element unless you want the font-family to be different, or if the element you want the same font on is a form element . i would also consider using group selectors for any other repeated styles like font-weight: bold; .
