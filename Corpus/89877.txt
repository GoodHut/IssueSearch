 URL : "https://github.com/zliuwd/learningnote/issues/4" TITLE : render vs reactdom.render BODY : render -------------------- 写在每一个组件里面，用来输出这个组件生成的html代码 app.jsx import react from 'react'; class app extends react.component { constructor props { super props ; this.state = { header: header from state... , content : content from state... } } render { return <div> <h1>{this.state.header}</h1> <h2>{this.state.content}</h2> </div> ; } } export default app; ---------------------------------- reactdom.render ---------------------------------- 写在最终输出的那个页面里面，例如html里面，整个app只是使用一次这个方法，用来形成最终的dom树。 main.js import react from 'react'; import reactdom from 'react-dom'; import app from './app.jsx'; reactdom.render <app />, document.getelementbyid 'app' ;
