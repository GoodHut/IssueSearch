 URL : "https://github.com/hlpassion/blog/issues/8" TITLE : mac install xgboost BODY : 做kaggle竞赛题，需要使用xgboost，正好自己电脑上没有装过，就直接来试一下。按照各个博客中的教程，加上自己尝试的过程，总结了一下步骤： - brew install gcc 接下来： 下载xgboost - git clone --recursive https://github.com/dmlc/xgboost - cd xgboost - cp make/config.mk ./config.mk - make -j4 如果make -j4 执行过程中报错 error: : errorunsupported option '-fopenmp' 首先执行 ls /usr/local/bin/ 查找其中的gcc 和 g++ 然后用vim 打开./config.mk 将被注释的 export cc export cxx 改成 export cc = /usr/local/bin/gcc-7 export cxx = /usr/local/bin/g++-7 然后在执行make -j4 最后在xgboost文件中进入python-package中执行python setup.py install即可
