 URL : "https://github.com/NicolasKogler/cranberry/issues/17" TITLE : abstract animation class BODY : abstract animation class goals having an abstract animation class that is able to render animations. subclasses override the create method and extract the frames out of the format they define e.g. gifanimation, cbaanimation, ... . approaches the class should do everything except for decoding the file it is given. each subclass overrides the pure virtual create method and calls createinternal which then will create the texture atlases and take care of the rest. requirements a class capable of packing multiple textures into one.
