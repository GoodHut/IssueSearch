 URL : "https://github.com/gatling/gatling/issues/3370" TITLE : extend stats in stats.json BODY : current gatling versions produce stats.json and global_stats.json files that are quite useful, but they still lack some information that would be very useful to have: - execution start time and end time or duration - definition of what the four different percentiles actually represent i.e. percentile4 == 99th percentile since that can vary between tests depending on the configuration much like groupn values that have name fields that describe the group
