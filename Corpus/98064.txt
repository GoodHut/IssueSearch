 URL : "https://github.com/gokrazy/gokrazy/issues/10" TITLE : implement squashfs support BODY : currently, we use a fat file system for both boot required and root convenient, because we already have an implementation . fat comes with a few disadvantages when used as a root file system: file names are truncated to 8.3 characters and symlinks are not supported. the most promising candidate for a simple root file system seems to be squashfs.
