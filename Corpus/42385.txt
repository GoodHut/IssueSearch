 URL : "https://github.com/ajhsu/blog/issues/44" TITLE : awesome babel plugins BODY : babel-plugin-module-alias a babel plugin to rewrite map, alias, resolve directories as different directories during the babel process. it's particularly useful when you have files you don't want to use with relative paths especially in big projects . https://www.npmjs.com/package/babel-plugin-module-alias example: js // instead of using this; import myutilfn from '../../../../utils/myutilfn'; // use that: import myutilfn from 'utils/myutilfn';
