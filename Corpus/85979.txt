 URL : "https://github.com/ethereum/solidity/issues/2918" TITLE : generic value range types BODY : support generic types of the form number<low..high ,precision > such that arithmetic operators on these types just naturally extend the ranges. example: function f number<0..2000> a, number<0..2000> b returns number<0..4000> { return a + b; } the type of the expression a + b is number<0..4000> . similarly, the type of a b would be number<0..4000000> . the precision parameter represents the number of decimal digits after the radix point and we have the natural rules: number<...,c> + number<...,d> -> number<...,max c, d > number<...,c> number<...,d> -> number<...,c + d>
