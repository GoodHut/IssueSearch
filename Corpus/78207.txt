 URL : "https://github.com/krukas/Mage2Gen/issues/53" TITLE : plugin naming convention BODY : mage2gen uses the whole namespace as path too the plugin class, but magento has a convention to use the following path vendor\module\plugin\{pluginclass}plugin for example if you write a plugin for the following class: aheadworks\blog\model\resourcemodel\postrepository mage2gen will generate the class on the following path vendor\module\plugin\aheadworks\blog\model\resourcemodel\postrepository but the magento convention is vendor\module\plugin\postrepositoryplugin
