 URL : "https://github.com/eclipse/microprofile-config/issues/204" TITLE : support for primitives BODY : presently, the following primitive types are not supported: - int - long - float - double - boolean these are the primitive versions for some of the classes defined in https://github.com/eclipse/microprofile-config/blob/master/spec/src/main/asciidoc/converters.asciidoc imho they should be supported, with the notion that if you use them they cannot be wrapped in optional.
