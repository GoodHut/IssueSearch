 URL : "https://github.com/haadcode/ipfs-log/issues/104" TITLE : add a cli tool BODY : it would be great to have a cli tool to manage logs. the basic commands would be: $ ipfs-log create qmfoo1 $ ipfs-log append qmfoo1 hello world qmfoo2 $ ipfs-log values qmfoo2 { payload: hello world , ... } $ ipfs-log create --id 'logb' qmfoo3 $ ipfs-log append qmfoo3 hi qmfoo4 $ ipfs-log join qmfoo2 qmfoo4 qmfoo5 $ ipfs-log values qmfoo5 --size 2 { payload: hello world , ... }, { payload: hi , ... } i would use yargs https://www.npmjs.com/package/yargs to wrap the commands. this one is up to grabs and if anyone wants to work on this, feel free to assign it to yourself or claim it here by saying so in a comment.
