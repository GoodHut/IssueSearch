 URL : "https://github.com/jmcnamara/XlsxWriter/issues/444" TITLE : dict as default parameter BODY : hi, in various methods of the worksheet class a default dict is provided in the method definitions. these can lead to problems as dicts are mutable. see: @convert_cell_args def insert_chart self, row, col, chart, options={} : the correct method definitions should be: @convert_cell_args def insert_chart self, row, col, chart, options=none : options = options or {} ... these happens in more places like: write_comment, set_columns, etc...
