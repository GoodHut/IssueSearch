 URL : "https://github.com/STayinloves/CUGBACM/issues/35" TITLE : f - mg loves gold BODY : 题意很简单，理解一下就是相连的一段金矿颜色不能有重复的。题目要求这样的一段一段的到底有多少段。 看到这个题我想到了stl里面的set。由于set插入元素之后可以使用count比较快的看看之前有没有加入过这个元素。我觉得很适合这个题目，所以解法就是从第一个金矿往后遍历，加入这一块之前用count看一下set里面有没有这种颜色的，如果没有就把这块加进去，如果有了就用clear把set清空然后计数的变量加一然后再将这一块插入set中。 代码： include <bits/stdc++.h> using namespace std; int t,n,k,i,s; int main { scanf %d ,&t ; while t-- { s=1; scanf %d ,&n ; set<int> m; for int i=1;i<=n;i++ { scanf %d ,&k ; if m.count k { s++; m.clear ; m.insert k ; } else m.insert k ; } printf %d ,s ; } return 0; }
