 URL : "https://github.com/SSteve/f-scott-fitzgerald_this-side-of-paradise/issues/2" TITLE : miscurled quotation marks BODY : use this regex to find quotation marks that should be reviewed: \s‘ a-z for example in chapter-1-1.xhtml : <p>“ ‘lo, amory.”</p> the single quote is curled in the wrong direction. because that is an elision, it should a right single quote, not a left single quote. there are at least several other instances of this issue in the text.
