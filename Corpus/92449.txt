 URL : "https://github.com/jgthms/bulma/issues/503" TITLE : wrap $colors in map merge BODY : overview of the problem this is about the bulma css framework description it would be great to be able to add more colours to the $colors map. a simple way to allow this is to wrap it in a map-merge function: $colors: map-merge $colors, ... this allows you to create your own map in an override before importing bulma: $colors: map-merge $colors, secondary: $secondary, $secondary-invert !default
