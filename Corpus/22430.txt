 URL : "https://github.com/upstage-org/limelight/issues/33" TITLE : layers for drawing BODY : in upstage there are 4 layers for drawing, which means that a player can select to draw in front of, between or behind avatars, or behind the backdrop. this is quite a useful feature, for example if you want to draw a curtain over everything, or have avatars in front of drawings, or reveal a drawing when a backdrop is removed. with the 4 layers, only one player at a time can draw in each layer. i don't know that this is necessarily an important feature - maybe it would be better to make it so that multiple players could draw at the same time in the same layer, if that's possible.
