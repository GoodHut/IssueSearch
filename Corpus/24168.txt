 URL : "https://github.com/disco-lang/disco/issues/57" TITLE : sets and multisets BODY : sets and multisets will have similar literal syntax as lists, except the delimiters will be { ... } for sets, and perhaps {+ ... +} for multisets. similarly to lists, they should support comprehensions though they will not support the same polynomial sequence syntax as lists . the interesting part is elimination. we will need a system to annotate functions with properties such as commutative, idempotent, identity element, etc. then the basic idea will be to have a primitive syntax reduce , such that reduce f when given an f : a -> a -> a will reduce a list, set, or multiset to a value of type a , as long as f has the appropriate properties.
