 URL : "https://github.com/inaturalist/INaturalistIOS/issues/406" TITLE : sounds in the ios app BODY : we’ll need to tackle a few things to implement sounds in the ios app: 1 sound playback of observations made externally, 2 add sounds to observations. one approach to adding sounds to the ios app would be via sharing from apps like voice notes on iphone. a problem with this approach is determining geolocation. the user would have to add the location manually, since i don’t believe voice notes provides geo data with the sound file being shared. this would be the simplest approach technically, since we wouldn’t have to add ui for creating and editing sound. another approach to adding sounds to the ios app would be via direct recording in the app. this is more work, since it requires more ui, but would allow us to capture geolocation at the time of audio capture.
