 URL : "https://github.com/tocodeil/unix1-course-examples/issues/68" TITLE : recursive glob - globstar BODY : there is a cool and almost hidden feature of bash 4 and up called globstar that allows you to create recursive globs, like this: will match any file under current dir & it's sub directories recursively .c will match any .c file under current dir & it's sub directories recursively / .c will match any .c file under any sub directory recursively , but in the current directory just remember that you need to enable it first with the following command: shopt -s globstar in the real world you will often use find for matching files recursively, but still, it's useful to know about it, because newer shells like zsh have it enabled by default, and many build tools use it a lot this is how i found out about it .
