 URL : "https://github.com/sikthehedgehog/indigo/issues/13" TITLE : add a virtual keyboard BODY : for when there isn't a keyboard connected... which i suppose is going to be the most common case. it would let you type text using the cursor. it should be possible to toggle the virtual keyboard at will. it may be a good idea to just have applications tell indigo when they want a virtual keyboard. some just use the keyboard for shortcuts e.g. the calculator which would render the virtual keyboard pointless. as for other apps, they'd need to handle when the virtual keyboard shows/hides to adjust their gui layout accordingly since the virtual keyboard would cover a significant portion of the screen . an alternative may be just to make it always available, though apps still will need to explicitly handle the need for gui layout changes.
