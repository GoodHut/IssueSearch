 URL : "https://github.com/emberjs/website/issues/2965" TITLE : add a webapp manifest BODY : api docs ships with the ember-web-app addon that takes of creating a webapp manifest per build. in case of the main website we'd have to create one manually. https://tomitm.github.io/appmanifest/ is an excellent tool to generate the manifest file. refer to https://github.com/ember-learn/ember-api-docs/blob/master/config/manifest.js for configuration values used on api docs.
