 URL : "https://github.com/jtassin/react-responsive-table/issues/9" TITLE : clash of component names BODY : this package will cause issues with other packages as it makes use of similar component names. for example the component <row> will class with the component <row> from react-material-responsive-grid https://www.npmjs.com/package/react-material-responsive-grid . also the <table> component will clash with the default material-ui component also called <table> http://www.material-ui.com/ /components/table .
