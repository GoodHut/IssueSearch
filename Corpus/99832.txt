 URL : "https://github.com/cerrno/shux/issues/16" TITLE : no external function calls BODY : right now, our language _cannot_ interface with any outside functions. we need to design the syntax to do this asap , in time for our hello world demo. i propose that we do something like the following for a print function wrapper : extern print string ; extern print_int int ; extern print_float float ; etc. and then we write c wrappers for them: void shux_print char str { printf %s , str ; } void shux_print_int int d { printf %d , i ; } void shux_print_float float f { printf %f , f ; } which we will then link with our library. this might be tricky for vectors though.
