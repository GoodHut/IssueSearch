 URL : "https://github.com/kerrpy/kerrpy-cython/issues/1" TITLE : port cuda code to cython BODY : we need to port existing cuda code from this repo https://github.com/kerrpy/kerrpy to cython. the components to port are the following: - runge-kutta solver https://github.com/kerrpy/kerrpy/blob/master/kerrpy/kernel/solvers.cu - initial conditions calculator and principal kernel https://github.com/kerrpy/kerrpy/blob/master/kerrpy/kernel/raytracer.cu - image transformation https://github.com/kerrpy/kerrpy/blob/master/kerrpy/kernel/image_transformation.cu - raytracer https://github.com/kerrpy/kerrpy/blob/master/kerrpy/kernel/raytracer.cu some of the pieces need openmp parallelization substitute cuda parallelisation. here http://cython.readthedocs.io/en/latest/src/userguide/parallelism.html is a guide on how to use cython openmp features.
