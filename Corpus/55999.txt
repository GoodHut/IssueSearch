 URL : "https://github.com/finchMFG/openGuild-CMS/issues/1" TITLE : make sure mongo starts BODY : hey, first of all, nice project, did had some trouble with the install on a fresh ovh cheap vps. the script hangs at the last command, start $appname as start is not a recognized command i went to execute the ./startserver.sh with an error thrown by future, looking like mongodb was not running. mongo was indeed not running. fixed with : echo unit description=high-performance, schema-free document-oriented database after=network.target service user=mongodb execstart=/usr/bin/mongod --quiet --config /etc/mongod.conf install wantedby=multi-user.target > /etc/systemd/system/mongodb.service systemctl start mongodb
