 URL : "https://github.com/bbc/hive-runner/issues/180" TITLE : avoid duplicate workers BODY : with 178 a hive is able to create workers while it is trying to kill other workers. there is a possibility, therefore, that a new worker may be created for a device while another worker for the same device still exists and is still running a test. there needs to be some safeguard to make sure that this cannot happen.
