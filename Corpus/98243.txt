 URL : "https://github.com/stovmascript/react-native-version/issues/18" TITLE : incorrect git tag selected BODY : i was investigating a similar issue i was having to https://github.com/stovmascript/react-native-version/issues/8 and i noticed a small problem when the git tag is amended https://github.com/stovmascript/react-native-version/blob/master/index.js l252 . git tag | tail -1 returns the last tag, but not in ref order. for example: $ git tag -l v0.1.0 v0.10.0 v0.10.1 v0.2.0 v0.3.0 v0.4.0 v0.5.0 v0.6.0 v0.7.0 v0.7.1 v0.7.2 v0.8.0 v0.9.0 git tag | tail -1 returns v0.9.0 , but the last tag is actually v0.10.1 . specifying the sort order gives the right tag: git tag -l --sort=v:refname
