 URL : "https://github.com/ralphbchance/prj-rev-bwfs-dasmoto/issues/1" TITLE : in html styles BODY : <h2 style= background-color:mediumspringgreen; >brushes</h2> the purpose here is to use your stylesheet with classes, ids and some element selectors to apply your styles. for example you could have given this a class of brushes <h2 class= brushes >brushes</h2> and then applied the style using this class in the css .brushes{ background-color: mediumspringgreen; } the idea here is that you can reuse classes. if you had 30 brush h2 elements and you wanted to change the color, your way would mean replacing the color 30 times. with the class you only have to change the color once.
