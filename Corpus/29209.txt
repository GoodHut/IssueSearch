 URL : "https://github.com/hatemogi/hangul-coding.org/issues/1" TITLE : julia 언어 예제 BODY : 안녕하세요 한글 코딩을 사랑하는 동지 ? 로써 반갑습니다. 저는 julia http://docs.julialang.org/en/stable/ 언어로 게임 밸런싱 시뮬레이션을 만드는데, 혼자하는 작업이고 게임의 소스코드를 보는게 아니라 ui나 기획서를 보면서 작업하다보니 한글로 코드를 짜게되는데 대표적인 예제는 아래와 같습니다. function 아이템강화비용 x::item ref = @where item_cost x.class , :rank .== x.rank, :enchant .== x.enchant cost = ref 1, :cost_currency return cost end function 아이템강화! ac::account, x::item if remove! ac, 아이템강화비용 x 아이템강화! x end end
