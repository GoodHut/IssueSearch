 URL : "https://github.com/arangodb/arangodb/issues/2375" TITLE : cors guide / example BODY : the foxx documentation makes no mention of cors. to send and receive cookies, the following line must be present in every route/controller res.headers 'access-control-allow-credentials' = true to send session id in header, add the following line to the login router. res.headers access-control-expose-headers = 'etag, content-encoding, content-length, location, server, x-arango-errors, x-arango-async-id, x-session-id' being able to set a default header would go a long way, but i don't know if that's possible.
