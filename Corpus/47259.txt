 URL : "https://github.com/DarwinJS/ChocoPackages/issues/39" TITLE : openssh: dsc / system BODY : package installation fails when installing the openssh package under dsc. the method gettempfilename used in the installer fails when run as system as the principal does not have a user-specific temp directory. i've replaced the method call in my simplified version of the install script with the following: powershell $params = @{ path = environment ::getenvironmentvariable 'temp', 'machine' childpath = system.io.path ::getrandomfilename } $path = join-path @params
