 URL : "https://github.com/gradle/gradle-native/issues/284" TITLE : support clang on windows BODY : this is a place to collect some issues with gradle's support for clang on windows. the clang for windows distribution does not include a c or c++ standard library implementation or any windows apis. the clang toolchain implementation should discover an implementation of these things, presumably the msvc implementation or maybe the mingw implementation, and provide this when compiling c or c++ source.
