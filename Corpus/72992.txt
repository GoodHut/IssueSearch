 URL : "https://github.com/2sComplement/fable-import-signalr/issues/1" TITLE : client to server example BODY : here's an additional example for two-way communication: // define the interfaces used for calls to client and server must match definitions on server type imyhubclient = abstract messagetoclient: string -> unit with get, set type imyhubserver = abstract messagetoserver: string -> unit with get, set type imyhubclientserver = abstract client: imyhubclient with get, set abstract server: imyhubserver with get, set type imyhubconnection = inherit signalr abstract myhub: imyhubclientserver // get the connection and hubproxy object let connection = globals.jquery.connection :> imyhubconnection // configure connection as needed connection.hub.logging <- true connection.hub.url <- http://localhost:8080/signalr let myhubproxy = connection.myhub // assign function to handle receiving message from server myhubproxy.client.messagetoclient <- fun message -> printfn client received message: %s message connection.hub.start // define function to use in client code to send message to server let sendmessage message = myhubproxy.server.messagetoserver message sendmessage hi from client
