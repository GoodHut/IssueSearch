 URL : "https://github.com/ku-nlp/jumanpp/issues/26" TITLE : rnn speedup ideas BODY : presently rnn information is computed for each left-node x beam x right-node pair triple. the core insight is that there exist many nodes with the same length -> identical segmentation paths inside the lattice, for which rnn computes exactly the same values multiple times. we can try to deduplicate the paths and compute rnn states only for unique ones. that should reduce greatly reduce computations done by rnn.
