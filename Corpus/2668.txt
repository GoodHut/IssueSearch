 URL : "https://github.com/charl26/Abstract-VM/issues/11" TITLE : command : div BODY : div: unstacks the first two values on the stack, divides them, then stacks the result. if the number of values on the stack is strictly inferior to 2, the program execution must stop with an error. moreover, if the divisor is equal to 0, the program execution must stop with an error too. chatting about floating point values is relevant a this point. if you don’t understand why, some will understand. the linked question is an open one, there’s no definitive answer.
