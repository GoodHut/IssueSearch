 URL : "https://github.com/CelticMinstrel/wesnoth/issues/391" TITLE : custom lua debug commands BODY : original submission https://gna.org/bugs/index.php?22609 by anonymous on 2014-09-11 currently we can call all global lua functions in debug mode like :lua some_function arg_1, ..., arg_n it would be nicer to be able to make custom debug commands by lua with wesnoth.debug_commands.my_new_command = function arg_1, ... , arg_n --somecode ... end which could then be called like my_new_command arg_1 ... arg_n
