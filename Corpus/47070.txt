 URL : "https://github.com/yuchenhou/elephant/issues/38" TITLE : node vector semantics BODY : problem the model can map each node to a node vector and use that vector to do prediction. however, we don't know the semantics of a node vector, i.e., we don't know the meaning of each item in a node vector. consequently, we don't know what information about each node the model has gathered and how it uses that information to do prediction. if we can't get a deeper understanding on how the model works, it's hard to explain why this model can outperform non-deep learning models. observations in word2vec https://www.tensorflow.org/tutorials/word2vec technique, differences of word vectors can reveal word vector semantics, e.g., verb tense. approach use node vector differences to reveal node vector semantics, e.g., node distance.
