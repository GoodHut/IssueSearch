 URL : "https://github.com/bandre-ucar/manage_externals/issues/14" TITLE : create generate_model tool BODY : create a tool to generate model description files based on the contents of an existing model description file and the working tree. functionality read an existing model description file and it's sub-externals. check the working copies on the file system, updating any modified repositories or branches in the model description. update any referenced sub-externals filenames to add the user specified prefix. write the self consistent set of new externals files. command line options --model - base model description file to use --prefix - prefix to use for output model description files, default 'user'. set to an empty string, '', to overwrite the existing files.
