 URL : "https://github.com/Doazeok/site/issues/5" TITLE : работа с api, ajax BODY : давай добавим на сайт такой функционал: с какого-то стороннего сайта будут забираться некие посты, и эти посты будут показываться на нашем сайте. запросы делаются с помощью api - по сути, это урл, который возвращает тебе не интернет страницу, как обычная ссылка, а json - особый формат данных по сути, просто словарь, переведнный в строку . например, есть словарь: var dict = { 'test': '1' } при переводе в json получим следующее: { test : 1 } т.е. по сути просто преобразование объекта в строку, которую можно отправить по сети в качестве данных. ------------------------------- запросы делаются асинхроно это значит страница не будет виснуть, дожидаясь ответа от стороннего сайта . делается это с помощью ajax - аякс в простонародье. в jquery есть удобная функция для ajax, почитай: https://www.w3schools.com/jquery/ajax_ajax.asp ----------------------------- наконец, само задание. есть сайт: https://jsonplaceholder.typicode.com/. это сайт который дает тестовые api ручки ссылки . например, /posts, возвращаюя посты. сайт прикольный, прямо там можно посмотреть что по ручкам отдается. надо: 1 создать новую страницу на нашем сайте для этих постов. 2 создать модель нокаутовскую 3 в нокаутовской модели будет функция, что-нибудь вроде getposts, которая будет делать ajax-запрос на этот сайт и получить посты ручка https://jsonplaceholder.typicode.com/posts . полученную инфу ты будешь сохранять в observablearray и выводить на страницу. вызываться метод при заходе на страницу. ручку https://jsonplaceholder.typicode.com/posts можешь прямо в браузере ради интереса проверить, посмотерть как выглядит возвращаемый json с постами.
