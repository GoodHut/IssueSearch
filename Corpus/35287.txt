 URL : "https://github.com/projecthamster/hamster-gtk/issues/189" TITLE : do not install tests BODY : when installing the app using python setup.py install --root= ${pkgdir} --optimize=1 , the tests will also be installed, even when calling sed s/find_packages /find_packages exclude= 'tests' / -i setup.py beforehand. this may lead to a conflict when installing another app that contains the same file e.g. /usr/lib/python3.6/site-packages/tests/helpers/__init__.py . default build log: https://paste.gnome.org/pgfkanom1 build log with tests excluded and removed from manifest.in : https://paste.gnome.org/pv04lfs9l
