 URL : "https://github.com/skulumani/system_setup/issues/13" TITLE : linux setup script BODY : for silent anaconda installation ~~~ wget http://repo.continuum.io/miniconda/miniconda3-3.7.0-linux-x86_64.sh -o ~/miniconda.sh bash ~/miniconda.sh -b -p $home/miniconda export path= $home/miniconda/bin:$path ~~~ also make sure to modify the path in all the dotfiles to point to ~/anaconda instead of ~/anaconda3
