 URL : "https://github.com/FAC10/week5-jajascript/issues/62" TITLE : use of module.exports BODY : in tflapi.js and tfllogic.js when you use module.exports you export an object with one key value pair, being the function created in that module. you can instead just export a single function, so rather than doing module.exports = { myfunction: myfunction }; <br>you can instead just use module.exports = myfunction <br>saves some characters in the long run :smile_cat:
