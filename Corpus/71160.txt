 URL : "https://github.com/betaflight/betaflight/issues/4226" TITLE : acc filtering is broken. BODY : after examining the startup code i found acc filtering to be broken. activateconfig calls setaccelerationfilter. acc.accsamplinginterval is used to set the filter. however, acc.samplinginterval will not have been configured yet. acc.samplinginterval is set by accinit accinit is called by sensorsaudtodetect it looks like the filter will never be initialised.
