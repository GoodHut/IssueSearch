 URL : "https://github.com/mozilla/taar-api/issues/16" TITLE : use taar's extra_data BODY : in mozilla/taar 51 we enabled the taar library to use additional data coming from the api caller. we need to make sure the taar-api passes this data around if received. the api could be invoked with something like: https://taar.dev.mozaws.net/api/recommendations/{uuid}/?locale=en-ca&platform=winnt instead of the plain https://taar.dev.mozaws.net/api/recommendations/{uuid} any extra url parameter needs to be sanitised and passed over as part of extra_data to taar.
