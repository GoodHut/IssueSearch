 URL : "https://github.com/xnau/list_field/issues/1" TITLE : general feature set 1.0 BODY : the idea is to provide a basic list functionality with the possibility of releasing something more advanced and featureful later. basic feature set: - single field that stores a list of items - columns are defined per field, or perhaps only one list field can be defined - column data types will be simple ones only: text, text-area, image, file, date, url, radio, checkboxes - user can add/delete/edit any line - possible to export the list data with the whole record via csv? no - possible to export the table data only via csv? - data is stored in separate table
