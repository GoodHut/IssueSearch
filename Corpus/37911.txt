 URL : "https://github.com/serbanghita/Mobile-Detect/issues/700" TITLE : new 3rd party module BODY : name, link and description of the module name : mobile detect for wordpress link : https://wordpress.org/plugins/tinywp-mobile-detect/ description : wordpress has a built-in function wp_is_mobile to detect mobile devices. there is at least one catch, though. it considers ipad ipad pro, and any tablet as a mobile. so, this 3rd party module changes the way wp_is_mobile works with the help of mobile detect php library!
