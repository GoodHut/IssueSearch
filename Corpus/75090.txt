 URL : "https://github.com/GeoscienceAustralia/PyRate/issues/148" TITLE : gdal installation issues BODY : gdal python bindings are available only for gdal versions: 1.5.0, 1.5.2, 1.6.0, 1.6.1, 1.7.0, 1.7.1, 1.8.1, 1.9.0, 1.9.1, 1.10.0, 1.11.0, 1.11.1, 1.11.2, 2.0.0, 2.0.1, 2.1.0, 2.1.3. make sure to install one of these versions. here is how to do it: curl -o http://download.osgeo.org/gdal/2.1.0/gdal-2.1.0.tar.gz tar -xzf gdal-2.1.0.tar.gz cd gdal-2.1.0/ ./configure --with-python sudo make install should capture this in instructions somewhere.
