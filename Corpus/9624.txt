 URL : "https://github.com/batonman/wk7CAPSTONE/issues/7" TITLE : a note about git BODY : in your repo you have included the file '.csscomb.json'. i assume it is something from a plugin for your editor. this should not be in your repo, it is unnecessary because it is not code for the actual project. in git, you can include a file called '.gitignore'. in that file, you can specify which files should _not_ be pushed into your git repo. this is helpful for hiding files such as the .csscomb.json. you can read more about gitignores here https://git-scm.com/docs/gitignore .
