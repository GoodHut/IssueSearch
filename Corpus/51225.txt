 URL : "https://github.com/cmakler/kgjs/issues/1" TITLE : add tsconfig.json file BODY : instead of using tsc command line flags, put them into package.json in the scripts section: scripts : { build : tsc src/ts/kg.ts --outfile kg.js --sourcemap }, and then run npm run build or make a tsconfig.json file: most of these lines are optional { compileroptions : { outfile : kg.js , sourcemap : true }, files : src/ts/kg.ts } and then run tsc there are also some potentially useful compiler options: { compileroptions : { noimplicitany : false, noimplicitreturns : true, noimplicitthis : true, strictnullchecks : true, noemitonerror : true, outfile : kg.js , sourcemap : true, }, files : src/ts/kg.ts }
