 URL : "https://github.com/Kotlin/anko/issues/343" TITLE : right to left support BODY : how do i support rtl layouts? 1. consider this layout: verticallayout { textview { text = whatewer textsize = ... } } 1. set android:supportsrtl= true in the manifest in application tag. 1. go to developer settings on the device and check force rtl layout direction . the textview should be aligned on the right, but it's not and documentation doesn't mention anything about supporting rtl.
