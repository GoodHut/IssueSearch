 URL : "https://github.com/jeffmon/foreclosure/issues/1" TITLE : revealing module pattern BODY : make sure you return an object with properties referencing the functions instead of writing out the function inside of the return object. ex: { myfunc1: myfunc1, myfunc2: myfunc2 } it's cleaner, and in addition it allows you to do object shorthand notation like so: { myfunc1, myfunc2 }
