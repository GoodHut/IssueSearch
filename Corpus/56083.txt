 URL : "https://github.com/w3c/strategy/issues/88" TITLE : media playback quality BODY : the media playback quality https://wicg.github.io/media-playback-quality/ specification extends the htmlvideoelement interface to add new features that can be used to detect the user perceived playback quality. the features were dropped from media source extensions https://www.w3.org/tr/media-source/ . such features would be useful to assess the user's perceived playback quality and alter adaptive streaming settings accordingly to maximize the quality of experience the specification is being proposed to the web incubator cg https://www.w3.org/community/wicg/
