 URL : "https://github.com/Gracij/Cminus-Parser2/issues/1" TITLE : how to use typespec BODY : typespec enumerates void, integer, array and is to be used on decl nodekinds. when we print out the parse tree each decl should display name followed by void , int , array , respectively see sample program output . in the grammar spec in the .y, decls of the form type id should have typespec array also see sample output . it may also be necessary to check parameters within the brackets, maybe by giving such decls child parameter decls.
