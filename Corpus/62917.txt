 URL : "https://github.com/adonisjs/adonis-framework/issues/560" TITLE : csrf token with angular BODY : hi! i'm setting up a new project with adonisjs at backend and angular 4 at frontend. stumbled with such a problem. adonisjs sends a cookie xsrf-token in the response to a client. by default angular adds this cookie in the next request at the same field named xsrf-token. but adonisjs only checks an input property of the request and the request headers. /node_modules/adonis-middleware/src/shield/index.js ... const csrftoken = request.input '_csrf' || request.header 'csrf-token' || request.header 'x-csrf-token' || request.header 'x-xsrf-token' ... so then it returns an error csrf token mismatch'. this can be fixed this way const csrftoken = request.input '_csrf' || request.header 'csrf-token' || request.header 'x-csrf-token' || request.header 'x-xsrf-token' || request.plaincookie 'xsrf-token' would you please add some additional checks over request cookies in the next release of adonishs? thank you.
