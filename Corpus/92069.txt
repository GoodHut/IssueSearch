 URL : "https://github.com/bibledit/bibledit/issues/462" TITLE : usfm 3 attributes support BODY : usfm 3.0 provides a general syntax for adding named attributes to character markers. attributes contribute descriptive metadata about the marked content. the addition of marker attributes is a means of extending the meta information contained within in a usfm text. usfm will formally provide descriptive attributes for a subset of character markers. each marker in this set will have a defined list of attributes, which are relevant to the overall purpose of the marker. https://ubsicap.github.io/usfm/attributes/index.html the issue calls for support for the attributes in usfm 3.0.
