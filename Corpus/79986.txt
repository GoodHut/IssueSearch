 URL : "https://github.com/cadeyrn/newtaboverride/issues/27" TITLE : support local html files BODY : for webextensions it's not possible to use the file:// protocol for local file access, therefore local files cannot be used as new tab page via custom url option. new tab override 8.0 will add support for uploading a local html file to the add-on's storage und use this as new tab page.
